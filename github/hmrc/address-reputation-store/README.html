address-reputation-store

This provides an abstraction for the MongoDB access that is used by both address-reputation-ingester and its dependent address-reputation-service.

Test Data

When running through service manager some test data is preloaded. You can see this test data under conf/data, for example these test addresses.

This test data is also pre-loaded in the Dev, QA and Staging instances of the service.
