http-core

http-core is a Scala micro library providing a set of common concerns and classes that help fulfillment of asynchronous HTTP calls.

It encapsulates some common concerns required for making HTTP calls on the HMRC Tax Platform, including:

  • Logging
  • Header Carrier
  • Http Transport
  • Core Http function interfaces

Adding to your app

Unless your project is a library that implements the HTTP transport and Core HTTP functions, this library should not be added to your project directly. Instead, include libraries that provide concrete implementaion of the http transport and Core http functions (ie: both http-verbs and http-verbs-play-25) to get the full functionality.

All examples are available here:hmrc/http-verbs-example
