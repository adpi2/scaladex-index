bulk-entity-streaming

Bulk Entity Streaming is a library that streams deliminator based data from a source of various types and provides an iterator or a list of converted entities. The motivation for this library is to provide applications with a consistent way of processing large sources of data (file, http) without the risk of encountering an out of memory exception, no matter how large the source data. The library expects a source function, a deliminator and a function to convert a 'row' (parsed based on the deliminator) of string-based data into an entity.

At the moment this library supports input sources of the following type:

  • Enumerator[Array[Byte]]
  • Future[Enumerator[Array[Byte]]]
  • Iterator[Char]
