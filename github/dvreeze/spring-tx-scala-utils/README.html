Scala wrapper for Spring TransactionTemplate

This is a trivial Scala wrapper around the Spring TransactionTemplate.

Although this wrapper is trivial, the result is that Scala code using a TransactionTemplate is less verbose when using the wrapper, and at least as easy to understand as code using the Spring TransactionTemplate directly.

Despite the declared dependency on a specific Spring 3 version (see the pom file), this wrapper should work with any 3+ version of Spring. When using this API with another version of Spring, make sure to exclude the currently used version of Spring in the pom file.
