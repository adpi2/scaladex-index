Grizzled-SLF4J: A simple, Scala-friendly front-end to SLF4J

The Grizzled SLF4J package provides a very thin Scala-friendly layer on top of the SLF4J (Simple Logging Façade for Java) API. It is released under a BSD license. Please see the web site for details.
