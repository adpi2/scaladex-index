Development environment

  1. Install Vagrant http://www.vagrantup.com/downloads.html
  2. cd to project directory
  3. vagrant plugin install vagrant-vbguest
  4. cp conf/application.conf.example conf/application.conf
  5. vagrant up (wait)
  6. vagrant ssh
  7. cd /vagrant
  8. sbt play run (wait)
