linking

To link to this library:

"de.sciss" %% "desktop" % v

Furthermore, if you want to support the Mac (OS X) platform, also add the following:

"de.sciss" %% "desktop-mac" % v

The current version v is "0.7.3".

building

Desktop currently builds against Scala 2.12, 2.11, 2.10, using sbt 0.13. To build sub project "desktop-mac", you need either OS X with Apple's EAWT extensions, or otherwise the dummy OrangeExtensions are used.

Some demos are available through sbt desktop/test:run. For Mac specific demos, sbt desktop-mac/test:run.
