FScapeJobs provides a simple OSC client to talk to the FScape audio signal processing toolbox, along with Scala configuration classes for most of its modules.

building

Builds with sbt 0.13 against Scala 2.11, 2.10. Depends on ScalaOSC. Standard sbt targets are clean, update, compile, package, doc, publish-local.

linking

Link to the following artifact:

"de.sciss" %% "fscapejobs" % v

The current version v is "1.5.0".
