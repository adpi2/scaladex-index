requirements

Builds with sbt 0.12.1 against Scala 2.10.0. Depends on Wolkenpumpe and ScalaCollider-Swing. Standard sbt targets are clean, update, compile, package, doc, publish-local.

creating an IntelliJ IDEA project

To develop the sources, if you haven't globally installed the sbt-idea plugin yet, create the following contents in ~/.sbt/plugins/build.sbt:

resolvers += "sbt-idea-repo" at "http://mpeltonen.github.com/maven/"

addSbtPlugin("com.github.mpeltonen" % "sbt-idea" % "1.0.0")

Then to create the IDEA project, run the following two commands from the xsbt shell:

> set ideaProjectName := "NuagesPompe"
> gen-idea
