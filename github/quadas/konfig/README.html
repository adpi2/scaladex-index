Konfig

A scala helper library for typesafe config

resolvers += Resolver.bintrayRepo("quadas", "maven")
libraryDependencies += "com.quadas" %% "konfig" % "0.1-M2"

bintray for now, we will publish to jcenter or maven central if enough people use it

examples

// application.conf
my-app {
    host = "0.0.0.0"
    port = 8080
}
// scala code
case class MyAppConfig(host: String, port: Int)
val config = ConfigFactory.load().read[MyAppConfig]("my-app")
