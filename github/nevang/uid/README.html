uid

uid is a Scala library for the generation and handling of 64-bit unique ids. See the project page and scaladoc for more details.

Setup

uid is available on the Sonatype and on the Maven Central repositories.

  • group id: gr.jkl
  • artifact id: uid_2.10
  • version: 1.1

Version 1.0, whithout value classes, is also available for Scala 2.10.0, 2.9.3-RC1, 2.9.2, 2.9.1-1, 2.9.1, 2.9.0-1, 2.9.0, 2.8.2 and 2.8.1.
