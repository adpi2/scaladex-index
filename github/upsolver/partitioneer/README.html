Partitioneer

This project wraps LevelDB factory as a function of Date, It allows to partition LevelDB as different files by the given date. The partition function can be any given function, A default implementation truncates the date by any given time (half an hour, 20 seconds, etc'). You can also use this project to partition other things. A default implementation of "Partition Repository" is available for LevelDB.

Use

libraryDependencies += "com.upsolver" %% "partitioneer" % "1.0.3"

Code Example

Import 

import com.upsolver.partitioneer

And run

// Create the partitioner

def onPartitionDemotion(db :DB ): Unit = {
  // Do stuff when the partition is not the latest...
  println("Partition " + db.toString + " Demotioned")
}

def onPartitionDelete(db: DB): Unit = {
  // Do stuff before partition is deleted
  println("Partition " + db.toString + " Deleted")
}

val fiveSeconds = 5L seconds
val twentySeconds = 20L seconds

val datePartitioner = new TruncateDatePartitioner(fiveSeconds, twentySeconds)
val partitioner = new DateTimePartitioner[DB](datePartitioner, new LevelDBPartitionRepository("dist/cache", DateTimeFormatter.ofPattern("yyyy-MM-dd HH_mm_ss")))

partitioner.listenToDelete(onPartitionDelete)
partitioner.listenToDemotion(onPartitionDemotion)

// Get partition
val db = partitioner.Get(LocalDateTime())
db.put("hello".toCharArray.map(_.toByte), "World".toCharArray.map(_.toByte))

Build

Clone and then run sbt assembly
