sri-vdom

Sri-vdom provides an alternate approach to using the dom vs sri-web. I personally like the style better since it gives you smaller methods and correctly handles namespacing.

Getting Started

Add the following dependency, and you're good to go. sri-vdom currently supports 2.11 and 2.12

libraryDependencies ++= Seq(
  "com.dispalt" %%% "sri-vdom" % "0.2.1-SNAPSHOT"
)
