OeScript

OeScript - Distributed, Asynchronous, Scalable

Status:

  • The OeScript software project is UNDER CONSTRUCTION and there is no usable product yet. Watch this repo starting Jan 1, 2015 as OeScript gets designed and built.

  • The sbt build system is in place now with full CI support via Travis CI and Bintray and Docker.

  • NOTE: Track the design and features on the OeScript Wiki.

An experimental Open Source re-implementation of OeScript using Scala and large-scale data processing software.

OeScript is server software with a tuplespace coordination language supporting "read", "write", "take", leases, and transactions, and with the additional feature of "attachments".

Notes about the library and tool dependencies and the architecture desicions are posted as issues.

Documentation is on the OeScript Wiki.
