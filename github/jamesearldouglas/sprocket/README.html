Sprocket is a collection of functional additions to the Java and Scala standard libraries.

Usage

Add Sprocket to build.sbt:

libraryDependencies += "com.earldouglas" %% "sprocket" % "0.1.0"

Examples

Builder

Call a void-returning method on an instance, and return the instance. This is useful with those pesky "setter" methods.

import com.earldouglas.sprocket.builder

val myJavaBean: MyJavaBean = new MyJavaBean() <+ { _.setFoo("foo") } <+ { _.setBar(42) }

Reader

Add map and flatMap to Function1 instances.

import com.earldouglas.sprocket._

val sq: Int => Int = x => x * x
val show: Any => String = x => x.toString

val sqThenShow: Int => String = sq map show

val x: String = sqThenShow(7) // the string "49"
val sq: Int => Int = x => x * x
val add: Int => Int => Int = x => y => x + y
val show: Any => String = x => x.toString

val sqThenAddThenShow: Int => String = for {
  squared <- sq
  added   <- add(squared)
  shown    = show(added)
} yield shown

val x: String = sqThenAddThenShow(7) // the string "56"

Close java.io.Closeable instances after use.

def cat: RandomAccessFile => Stream[String] =
  { x =>
    Option(x.readLine()) match {
      case None       => empty
      case Some(line) => cons(line, cat(x))
    }
  }

val head =
  for {
    stream <- cat
    lines  = stream.take(5)
  } yield lines.toList

val x = head(new RandomAccessFile("foo.txt")) // the first five lines of foo.txt
