yorel

Yorel is a simple Scala library with some types that I found very useful on different kind of projects.

Those are usually high-kind types that receives the type of exception they are going to throw under certain conditions. Also, each type has an Implicits object that contains some useful implicit primitives to transform between primitives and the smart types.

Installing

The library is on Sonatype Nexus, so you only need to add the repository if you already didn't. Currently Yorel is only compiled and distributed for Scala 2.11.

resolvers ++= Seq(Resolver.sonatypeRepo("releases"))

Current version is:

lazy val Yorel = "com.github.kmruiz" %% "yorel" % "0.0.1"

Index

BoundString[E]

Throws an exception of type 'E' when the provided String is null or empty. This is a common type for every String value, because null or empty values usually are forbidden.

case class User(name: BoundString[IllegalUserNameException])

Email[E]

Throws an exception of type 'E' when the provided String is not a valid email.

NonNull[E, T]

Throws an exception of type 'E' when the provided element of type 'T' is null.

case class User(name: NonNull[IllegalUserNameException, String])

Password

Value Object that contains logic for BCrypted passwords check and generation. It requires a bcrypt salt for creating new passwords.

implicit val TestBcryptSalt = BcryptSalt(BCrypt.gensalt(4))

val myPassword = Password("123456")

UUID

Simple object that generates new UUIDs and throws an error when an existing invalid uuid is provided.

GenericException

throwableOf[E <: Throwable](value: String, cause: Throwable)

Generates a Throwable of type E with the provided parameters. This object is used around the code for throwing custom exceptions.

MonadTransformers

Object with implicit methods that changes the type of different monads maintaining the state. Currently there is only a bind from T to Future[T].
