spark-tests

Utilities for writing tests that use Apache Spark.

  • give your tests fresh SparkContexts
    • per-Suite (a thin wrapper around holdenk/spark-testing-base's SharedSparkContext)
    • or per-case
  • make an RDD with specific elements in specific partitions,
  • create and cleaning up temporary files and directories
  • verify the number of Spark jobs that have been run
