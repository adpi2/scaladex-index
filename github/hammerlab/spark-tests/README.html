spark-tests

Utilities for writing tests that use Apache Spark.

  • Give your tests fresh SparkContexts:
    • per-Suite: SparkSuite (a thin wrapper around holdenk/spark-testing-base's SharedSparkContext)
    • per-test-case: PerCaseSuite.
  • KryoSparkSuite: SparkSuite implementation that provides hooks for kryo-registration.
    • Useful for subclassing once per-project and filling in that project's default Kryo registrar, then having concrete tests subclass that.
    • cf. hammerlab/guacamole, hammerlab/pageant.
  • rdd.Util: make an RDD with specific elements in specific partitions.
  • NumJobsUtil: verify the number of Spark jobs that have been run.
  • RDDSerialization: interface that allows for verifying that performing a serialization+deserialization round-trip on an RDD results in the same RDD.
