spark-util

Low-level helpers for Spark libraries and tests, currently:

  • KeyPartitioner: assigns keys to partitions based on their value (if an Int) or the first value of a 2- or 3-tuple
  • type aliases for Ints: PartitionIndex and NumPartitions
  • Conf: parses (comma-delimited) lists of properties files from a provided argument (as well as the $SPARK_PROPERTIES_FILES env var) and builds a SparkConf with those properties set.
