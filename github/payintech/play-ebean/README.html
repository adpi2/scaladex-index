play-ebean

This module provides Ebean support for Play Framework 2.5 and superior.

About this project

This project was forked from the original repository playframework/play-ebean.

Configure your Play application

project/plugin.sbt

resolvers += Resolver.sonatypeRepo("releases")

addSbtPlugin("com.payintech" % "sbt-play-ebean" % "YY.MM")

You have to replace YY.MM with available release you want to use (see Releases tab).

build.sbt

resolvers += Resolver.sonatypeRepo("releases")
