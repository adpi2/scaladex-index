IODB - database engine for blockchains

IODB is persistent key-value store. It is inspired by RocksDB . It has:

  • ordered key-value store
  • snapshots with rollbacks
  • multi-threaded compaction
  • batch imports

Examples

Code examples are in src/test/scala/examples folder.

Compile

  • Checkout IODB
  • Install SBT
  • Compile IODB: sbt publish
