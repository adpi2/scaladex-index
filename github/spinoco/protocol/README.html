protocol

This library contains various communication protocol decoders and encoders for scala.

All the codecs are implemented with scodec.

Following list of protocols is implemented:

  • STUN protocol according to RFC5389 and RFC5245
  • WebSocket protocol according to RFC6455
  • HTTP 1.1 protocol headers (Request/Response)
  • RTP and RTCP RFC3550
  • SDP RFC4566
  • MGCP RFC3435
