Scalamachine

Scalamachine is a port of Basho's Webmachine to Scala. It is pluggable into different server frameworks:

  • Netty
  • Finagle
  • Lift
  • Servlets

"You can call Webmachine a REST toolkit if you like, and we won't argue with you." - Webmachine Wiki

For more info check out the website
