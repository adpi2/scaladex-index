sbt-dependencies-tests

Sample project for testing sbt-dependencies plugin

Latest version:

libraryDependencies += "com.47deg" %% "sbt-dependencies-test" % "0.1.22"

How to test it

  1. Create a GitHub token with repo permissions
  2. In the project root:
$ sbt -DgithubToken=XXXX
> showDependencyUpdates
> updateDependencyIssues
