scala-ebook-clipper

A Scala library for extracting contents(TOC, figures, metadata, cover) from ebooks(PDF,).

Based on pdfbox.

Add the dependency

  • SBT:

    libraryDependencies += "com.transgee" %% "scala-ebook-clipper" % "1.0.0"

  • Maven:

    <dependency>
  <groupId>com.transgee</groupId>
  <artifactId>scala-ebook-clipper_2.11</artifactId>
  <version>1.0.0</version>
</dependency>

Usage

  • Scala

    import com.transgee.ebook._

// close automatically
withEbook("some/doc.pdf"){ ebook =>
  ebook.cover(toSomeOutputStream)
  val metadata: Metadata = ebook.metadata()
  val toc: Toc = ebook.toc()
  val figures: Iterator[Figure] = ebook.figures()
}

// close manually
val ebook = Ebook("some/doc.pdf")
try {
  ebook.cover(toSomeOutputStream)
  // ...
} finally {
  ebook.close()
}
