jvm-fork

Helpers to launch forked JVMs

Quick start

Add the following to your build.sbt

resolvers += Resolver.sonatypeRepo("snapshots")

libraryDependencies += "sh.jove" %% "jvm-fork" % "0.1.1"

Examples

import jove.jvmfork.Fork

val proc = Fork(new File("."), Nil, classPath, "mainClass", args)
// proc has type java.lang.Process

// It can be waited for with
proc.waitFor()
val retCode = proc.exitValue()
