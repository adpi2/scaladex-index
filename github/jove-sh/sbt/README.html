sbt ivy, classpath, io

The ivy sub-project of sbt doesn't really depend on the other sub-projects of sbt (but for io). Here it is for your convinience, just it and io.

The same applies to the classpath project, which provides ClassLoader and class path related utilities. This branch also contains a version of it, lighter then the original though.

Usage

Add to your build.sbt

libraryDependencies ++= Seq(
  // Or just one of these two if you need only one
  "sh.jove.sbt" %% "ivy" % "0.13.8-SNAPSHOT",
  "sh.jove.sbt" %% "classpath" % "0.13.8-SNAPSHOT"
)

Then see the APIs in the jove.sbt namespace.

Among others, the following changes were made compared to the sbt code:

  • the Logger class is the one from slf4j, instead of being a sbt specific one,

  • the whole code was moved to the jove.sbt namespace instead of just sbt, to avoid conflicts with the original sbt namespace.

Built and published for both Scala 2.10 and 2.11.
