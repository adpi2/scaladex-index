Flyway by Boxfuse

Database Migrations Made Easy.

Evolve your database schema easily and reliably across all your instances.

Simple, focused and powerful.

Works on

Windows, Mac OSX, Linux, Java and Android

Supported build tools

Maven, Gradle, Ant and SBT

Supported databases

Oracle, SQL Server, SQL Azure, DB2, DB2 z/OS, MySQL, MariaDB, Google Cloud SQL, PostgreSQL, Redshift, Vertica, EnterpriseDB, H2, Hsql, Derby, SQLite

Third party plugins

Grails, Play!, Grunt, Griffon, Ninja, ...

Documentation

https://flywaydb.org

How to contribute

https://flywaydb.org/documentation/contribute/
