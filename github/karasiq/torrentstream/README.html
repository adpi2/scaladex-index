TorrentStream

TorrentStream is a torrent streaming engine, written in Scala and based on akka-streams.

How to use

  • As an application
    • Start server with sbt run, or use precompiled package
    • Open http://127.0.0.1:8901 in browser
  • As a library
    • libraryDependencies += "com.github.karasiq" %% "bittorrent" % "1.0.6"
