Snowplow Scala Tracker

Overview

Add analytics to your Scala, Akka and Play apps and servers using the Snowplow event tracker for Scala.

Quickstart

Assuming git, Vagrant and VirtualBox installed:

 host$ git clone https://github.com/snowplow/snowplow-scala-tracker.git
 host$ cd snowplow-scala-tracker
 host$ vagrant up && vagrant ssh
guest$ cd /vagrant
guest$ sbt test
