freestyle-rpc

Simple RPC with Freestyle

Installation

Add the following dependency to your project's build file.

For Scala 2.11.x and 2.12.x:

"io.frees" %% "frees-rpc" % "0.0.3"

Or, if using Scala.js (0.6.x):

"io.frees" %%% "frees-rpc" % "0.0.3"

Demo

See freestyle-rpc-examples repo.
