Digital Ocean Client

A driver for V2 of the Digital Ocean API written in Scala on top of Netty and RxJava.

Install

Add the following to your Build.scala

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "digital-ocean" % "0.8.0"

Usage

To initialize the digital ocean client simply pass your digital ocean auth token to the DigitalOceanClient case class.

val doClient = DigitalOceanClient(doAuthToken)

All valid endpoints are simply just mapped to methods on the DigitalOceanClient instance.
