Conveyance

Conveyance is a system for building composable and asynchronous pipelines to process data. The two main abstractions in this library are asynchronous streams called Conduits and asynchronous values called Boxes.

Conduits:

Conduit
  .from[Throwable, Int](Seq(1, 2, 3, 4, 5, 6, 7))
  .map(_ * 2)
  .flatMap({ n => Conduit.of(n * 2) })
  .open(Pump(
    onNext      = (n) => println(n),
    onError     = (_) => println("Something went wrong."),
    onCompleted = ()  => println("Done!")
  ))

Boxes:

Box
  .of[Throwable, Int](42)
  .map(_ * 2)
  .flatMap({ n => Box.of(n * 2) })
  .open(Conveyor(
    onReceive = (n) => println(n),
    onError   = (e) => println("Something went wrong.")
  ))
