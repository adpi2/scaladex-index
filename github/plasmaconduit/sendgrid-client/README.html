SendGrid Scala Client

A library to make SendGrid api calls on top of plasmaconduit's rxnetty library.

Installation

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "sendgrid-client" % "0.4.0"

Usage

import com.plasmaconduit.sendgrid.SendGridClient
import com.plasmaconduit.sendgrid.SendGridEmail
import com.plasmaconduit.sendgrid.EmailUser

val recipient = EmailUser("agro@appdation.com", "Agro")
val from = EmailUser("info@appdation.com", "Appdation")
val email = SendGridEmail("Welcome to Appdation!", Some("We're really happy you're here!", None))

SendGridClient(SENDGRID_API_USER, SENDGRID_API_KEY)
      .sendMailTo(recipient, from, email)
      .map(response => {
        println("Send mail response: " + response.status.statusCode)
      })

You can also use the SMTPOptions object that is supplied to a SendGridExtras object to define things like templates and tracking to individual emails.
