Json Pointer

A very simple implementation of JSON Pointers ( http://tools.ietf.org/html/rfc6901 ) in scala.

Install

Add the following to your Build.scala

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "json-pointer" % "0.3.0"

Usage

Some JsonPointer query examples:

import com.plasmaconduit.json._
import com.plasmaconduit.jsonpointer._

val subject = JsObject(
  "name"      -> "Joseph",
  "age"       -> 26,
  "addresses" -> JsArray(
    JsObject("state" -> "WA"),
    JsObject("state" -> "CA"),
    JsObject("state" -> "GA")
  )
)

val name = for {
  pointer <- JsonPointer("/name")
  value   <- pointer.query(subject)
  string  <- value.as[String]
} yield string

val age = for {
  pointer <- JsonPointer("/age")
  value   <- pointer.query(subject)
  number  <- value.as[Int]
}

val secondState = for {
  pointer <- JsonPointer("/addresses/1/state")
  value   <- pointer.query(subject)
  string  <- value.as[String]
} yield string
