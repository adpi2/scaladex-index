JWT

A scala implementation of JSON Web Signatures.

This was developed against the draft-ietf-oauth-json-web-token-25 specification. Currently only supports a small subset.

Install

Add the following to your Build.scala

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "jwt" % "0.17.0"
