Algebro

A suite of traits that describe a small subset of the Haskell typeclass hierarchy in Scala.

Defines the following lawful traits:

  • SemiGroup
  • Monoid
  • Functor
  • Applicative
  • Monad
  • Comonad

Also implements the following combinators:

  • application
  • constant
  • fix
  • flip
  • identity

Install

Add the following to your Build.scala

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "algebro" % "0.1.0"
