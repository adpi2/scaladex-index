Try To Observable

This is a super simple convenience library for converting Try[A] values to Observable[A] values

Installation

resolvers += "Plasma Conduit Repository" at "http://dl.bintray.com/plasmaconduit/releases",
libraryDependencies += "com.plasmaconduit" %% "try-to-observable" % "0.2.0"

Usage

import com.plasmaconduit.trytoobservable._
import scala.util.Success

val observable = TryToObservable(Success("woo, observables!"))
