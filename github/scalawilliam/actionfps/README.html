Quickstart

More detail at the ActionFPS Development Guide.

  1. Install Oracle JDK 8 or OpenJDK 8.
  2. Install SBT (Mac, Windows, Linux).
  3. To make changes, in command line execute:
    sbt web/run
  4. To make changes involving stats comuptations etc, use:
    sbt 'web/run -Dfull.provider=normal'
