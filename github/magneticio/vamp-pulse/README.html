Vamp Pulse

Deprecated since Vamp 0.8.0 version. Preferred way to collect, aggregate and visualize logs and metrics is to use Elasticsearch, Logstash and Kibana.

Vamp Pulse is an aggregation engine which allows you to store and aggregate events in Elasticsearch using either REST calls, SSE or Kafka feeds. It works together with Vamp Router and Vamp Core

building and running

Clone this repo and use SBT to run it. You specificy your specific config file based on reference.conf and optionally your logback.xml file:

sbt -Dconfig.file=conf/application.conf -Dlogback.configurationFile=conf/logback.xml run

For more documentation on installing and using Vamp Pulse check http://vamp.io
