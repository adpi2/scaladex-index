I made a RPN calculator a while ago and I use it all the time. I figured I would share it with the world.

You can run it with coursier or with sbt.

coursier launch com.jarrodwb:rpn-calculator_2.12:0.1

or

coursier bootstrap com.jarrodwb:rpn-calculator_2.12:0.1 -o rpn
./rpn

or

sbt run

There is also a gui that I don't use but feel free to play around with it.

coursier bootstrap com.jarrodwb:rpn-calculator_2.12:0.1 -M rpn.gui.Gui -o rpn-gui

Inspect Cli.scala in the source code to get a feel of how to use the program. I write a tutorial if there is enough interest.
