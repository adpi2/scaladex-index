TimeSeries runtime for FlowScript

Custom runtime for FlowScript with functions working on Time Series data.

Features

This runtime adds the following functionality:

  • Time Series type (SeriesValue)
  • Scalar operation
  • Rolling window
  • Difference

Quick start

Add the following dependency to the build.sbt

libraryDependencies += "io.github.carldata" %% "timeseries-flow" % "0.3.0"
