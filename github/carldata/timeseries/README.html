Welcome to the Time Series library

Library for processing Time Series.

Quick start

Add the following dependency to the build.sbt

libraryDependencies += "io.github.carldata" %% "timeseries" % "0.3.0"

Running benchmarks

sbt -mem 4000 run

Features

  • Basic functionality
    • Slicing series
    • Map, fold and filter
    • Integration
    • Differentiation
    • groupBy
    • Rolling window
    • Resampling
  • Calculate statistics
    • min, max
    • mean, variance and standard deviation
  • IO
    • Read data from CSV string
  • Advanced functionality
    • Finding sessions (periods of activity)
    • Finding patterns
  • Anomaly detection
    • Outliers
