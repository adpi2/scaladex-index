Checkmate

Minimal, scala.js-compatible property-based testing support. This runnable example demonstrates keys features.

SBT

JVM: "com.github.benhutchison" %% "checkmate" % "0.1"

JS: "com.github.benhutchison" %%% "checkmate" % "0.1"

Built for Scala 2.11 only.

Motivation

This library is an alternative to ScalaCheck, with which I'm a little dissatified for various reasons:

  • ScalaCheck's generators return Option[T], not T values, which looks like unneeded complexity to me.
  • It's default Double generator is nearly useless ATM due to a poor distribution and not yet fixed.
  • Release 1.11 made breaking changes that seemed entirely avoidable and obtructed my use of it
