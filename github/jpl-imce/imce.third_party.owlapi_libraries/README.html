IMCE Third-Party OWLAPI Libraries

This project builds an aggregate of the OWL API published by several organizations, including:

  • net.sourceforge.owlapi The official OWL API as the de-facto reference implementation of the W3C OWL Functional Syntax Specification

  • org.openrdf.sesame The Eclipse RDF4J Sesame implementation of W3C Resource Description Framework, RDF
