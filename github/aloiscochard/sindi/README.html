Sindi

Sindi is a lightweight library easing the configuration and design of Scala applications in a safe way.

The library is composed of two independent modules:

  • sindi-core: IoC container with a static binding process (using compile-time implicits resolution)
  • sindi-config: Safe configuration system integrating the typesafe-config library

You'll find more information on the website, and support on the dedicated mailing list:

  • user guide: http://aloiscochard.github.com/sindi
  • mailling list: http://groups.google.com/group/sindi-users
