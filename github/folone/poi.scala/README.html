Setup

SBT:

libraryDependencies += "info.folone" %% "poi-scala" % "0.15"

Maven:

<dependency>
    <groupId>info.folone</groupId>
    <artifactId>poi-scala_${scala.version}</artifactId>
    <version>0.14</version>
</dependency>
